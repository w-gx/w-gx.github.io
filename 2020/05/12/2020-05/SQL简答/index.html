Q1：数据库管理系统的主要功能

  1. 数据定义功能
  2. 数据组织，存储和管理
  3. 数据操纵功能
  4. 数据库的事务管理和运行管理
  5. 数据库的建立和维护功能
  6. 其他功能

Q2：数据库系统的特点

  1. 数据结构化
  2. 数据的共享性高，冗余度低，易扩充
  3. 数据独立性高
  4. 数据由DBMS统一管理和控制

Q3：DBA职责

  1. 决定数据库中的信息内容和结构
  2. 决定数据库的存储结构和存取策略
  3. 定义数据的安全性要求和完整性约束条件
  4. 监控数据库的使用和运行
  5. 数据库的改进重组和重构

Q4：SQL特点

  1. 综合统一
  2. 高度非过程化
  3. 面向集合得操作方式
  4. 以同一种语法结构提供多种使用方式
    - 自含式
    - 嵌入式
  5. 语言简洁，易学易用

Q5:视图的作用

  1. 视图可以简化用户操作
  2. 视图可以使用户以多种角度看待统一数据
  3. 视图对重构数据库提供了一定程度的逻辑独立性
  4. 视图能够对机密数据提供安全性保护
  5. 适当利用视图可以清晰的表达查询

Q6：数据库安全性控制

  1. 用户身份鉴别
  2. 多层存取控制
    - 自主存取控制（DAC）
    - 强制存取控制（MAC）
  3. 审计
  4. 视图机制
  5. 数据加密

Q7：数据库设计的6个阶段

  1. 需求分析
  2. 概念结构设计
  3. 逻辑结构设计
  4. 物理结构设计
  5. 数据库实施
  6. 数据库运行和维护

Q8：数据字典的主要内容

  1. 数据项
  2. 数据结构
  3. 数据流
  4. 数据存储
  5. 处理过程
